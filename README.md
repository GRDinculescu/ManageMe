# ManageMe

### Objetivo

El objetivo es proporcionar a las tiendas, tanto pequeñas como grandes, un control fácil, rápido y efectivo sobre los productos.
Puesto que entendemos que en una tienda compran y venden productos constantemente, nos enfocaremos en hacer el proceso de gestión de inventario lo más fácil y rápido posible, claro, sin que eso quite funcionalidad.

### Realidad

Un Proyecto de clase de Interfaces para hacer una aplicacion de gestion de inventario
